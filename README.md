# ConsentApp Project

ConsentApp est une application permettant de gérer des consentements entre utilisateurs, avec un système de crédits pour créer des consentements. Le projet est divisé en deux parties :

- **app-frontend** : Application mobile React Native pour les utilisateurs.
- **app-backend** : API Node.js/Express pour gérer les utilisateurs, consentements, paiements Stripe, et notifications.

## Structure du projet
- `app-frontend/` : Frontend React Native.
- `app-backend/` : Backend Node.js/Express.

## Prérequis
- Node.js (v18 ou supérieur)
- Expo CLI (pour le frontend)
- PostgreSQL (pour la base de données)
- Compte Stripe (pour les paiements)
